In this project, I implement a 2D heat diffusion equation, based off the CUDA tutorial below. I adapted it to OpenCL using what we've learned so far. 

https://enccs.github.io/OpenACC-CUDA-beginners/2.02_cuda-heat-equation/


To run: 
make 
make run

assignment.exe usage (all optional): 
--platform n --width w --height h --secs s --a diffusion_constant
